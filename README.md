## 🛠️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/MuhammadShayan8401/shayan-portfolio.git
```

Navigate to project folder:

```bash
cd shayan-portfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open in browser:

```
http://localhost:3000
```
